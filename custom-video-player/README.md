# Custom Video Player

# Introduction
- HTML vidoe element를 이용하여 커스터마이징한 동영상 플레이어
- 개발기간 : 2020.12.14(1일)

- HTML
- CSS
- JavaScript
- Git, Github

# Features
- 동영상 재생(play), 중지(pause)
- 동영상 정지(stop)
- 동영상 재생바 변화
- 동영상 시간 (초:분) 표시
