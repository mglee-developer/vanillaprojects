# Features
- 영화 선택
- 좌석 선택 및 해제
- 좌석에 따라 금액 변경
- 좌석의 수와 금액 변경
- 영화, 좌석과 금액이 local storage에 저장
- 새로고침 시, 저장된 데이터 출력
